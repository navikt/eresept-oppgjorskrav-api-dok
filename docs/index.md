# Oversikt

Helserefusjonsløsningene KUHR (Kontroll og Utbetaling av Helserefusjoner), Eresept OK (OppgjørsKontroll) og Frikortløsningen
tilbyr API-baserte tjenester for samhandling med behandlere, apotek og andre aktører i helsesektoren som har avtale med HELFO om
direkte oppgjør.

Disse sidene beskriver tjenestene til Eresept OK.

## Tjenestene

Eresept OKs tjeneste for oppgjørskontroll er beskrevet her: [Eresept OK oppgjørskrav](oppgjorskontroll/eresept_ok_oppgjorskrav.md)
