# M22


Endepunkt for å hente oppgjorsresultat (M22) for et oppgjørskrav.

| Felt                     | Verdi                       |
|--------------------------|-----------------------------|
| path                     | /oppgjorskrav/m22           |
| query string parameter   | `oppgjorsnr`: et oppgjorsnr |
| method                   | GET                         |
| content type             | application/xml             |


| Status | Response  | Forklaring                                         |
|--------|-----------|----------------------------------------------------|
| 200    | M22 (XML) | M22 er tilgjengelig og returneres                  |
| 204    |           | M22 er ikke produsert og ingen respons returneres  |
| 400    |           | Oppgjøret finnes ikke                              |
| 451    |           | Direkteoppgjørsavtale er ikke på plass             |
