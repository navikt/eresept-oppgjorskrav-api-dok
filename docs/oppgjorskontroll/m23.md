# M23


Endepunkt for å hente utbetalingsresultat (M23) for et oppgjørskrav.

| Felt                    | Verdi                       |
|-------------------------|-----------------------------|
| path                    | /oppgjorskrav/m23           |
| query string parameter  | `oppgjorsnr`: et oppgjorsnr |
| method                  | GET                         |
| content type            | application/xml             |



| Status | Response  | Forklaring                                        |
|--------|-----------|---------------------------------------------------|
| 200    | M23 (XML) | M23 er tilgjengelig og returneres                 |
| 204    |           | M23 er ikke produsert og ingen respons returneres |
| 400    |           | Oppgjøret finnes ikke                             |
| 451    |           | Direkteoppgjørsavtale er ikke på plass            |
